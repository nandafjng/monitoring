Konfigurasi sumber data

Hubungkan Google Sheets

Dashboard ini membaca data dari Google Sheets yang kamu publish. Ikuti langkah berikut sekali saja — setelah itu tinggal update sheet-nya.

1

Buat Google Sheets dengan kolom ini (baris 1 = header)

week  |  cod_total  |  cod_success  |  cod_return  |  cod_reject
pp_total  |  pp_return  |  aov_cod  |  aov_pp

Contoh isi baris: W14-Apr, 950, 810, 95, 45, 1800, 180, 185000, 230000

2

Publish sheet sebagai CSV

Di Google Sheets: File → Share → Publish to web
Pilih sheet yang berisi data → pilih format CSV → klik Publish → copy URL-nya.

3

Paste URL di sini

URL akan disimpan di browser ini. Setiap orang yang buka dashboard perlu paste URL sekali.
